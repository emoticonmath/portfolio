Project: Scratch Game

Class: CSE (Computer Science and Engineering)

Duration: August 29, 2017 - September 8, 2017

Objective: To create a game in Scratch

Type of Project: Partner Project


Over the duration of the Scratch project, our goal was to create a game similar to an Inifinite runner, such as the dino run game on Google when you search for something with no Internet connection. This developed into the idea of Super Princess Ally, a game in which the player takes control as the protagonist of the game, and saves her love Prince Bill as he is kidnapped by the evil King Maticon. The game and code are viewable here. The game plays as follows; you, as Princess Dove, need to beat both levels by dodging and jumping over Goombas and cacti using the space bar, and after completing these levels, you save Prince Bill from the shackles of King Maticon.


Project: Pokemon Hangman Python Project

Class: CSE (Computer Science and Engineering)

Duration: January 2018 - Feburary 2018

Objective: To create a hangman game in the Python Programming Language.

Type of Project: Partner Project


The Hangman Project was a project in Computer Science, in which our goal was to write code for a Hangman game in Python. Using Cloud9, we made a Hangman game regarding charcacters from Pokemon that can be played here. To play the game, open the Hangman folder, and open YasarNguyenHangman.py. To play the game, open the bash terminal, then type in ipython, then run the code with the command %run Hangman/YasarNguyenHangman.py to play the game. You can downloaded the game here.
